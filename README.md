# Guitar Pasal

Welcome to the **Guitar Pasal** project repository!

"Pasal" means shop or store, and this project is dedicated to building a platform for guitar enthusiasts. Whether you're looking for new gear, managing inventory, or just exploring riffs, Guitar Pasal aims to be your go-to solution.

## Features
- **Inventory Management**: Track guitars, strings, amps, and accessories.
- **Neon Riffs**: A dedicated section for sharing and learning iconic guitar riffs.
- **User-Friendly Interface**: Designed for musicians, by musicians.
